Laptop

A laptop, also known as a laptop, is a personal computer that makes it easy to carry and work in different locations and terrain. Laptops have been designed to be fully functional like a desktop computer, which means they can run the same software and open the same files as a desktop computer. However, some types of laptops, such as Netbooks, leave out some functions to be more portable.
